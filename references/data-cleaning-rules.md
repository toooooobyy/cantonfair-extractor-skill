# 数据清洗规则

## 概述

数据清洗规则经过10个分类、1191家企业的实战验证，覆盖了广交会平台常见的各种脏数据模式。

清洗脚本：`scripts/04_process_data.py`

## 企业网站 (clean_website)

### 清洗流程

```
原始值 → 去空白 → 修复中文冒号 → 过滤无效值 → 检查非企业网站
       → 检查邮箱误填 → 检查公司名误填 → 补全http://前缀 → 域名小写化
```

### 详细规则

1. **去除空白字符**：`re.sub(r'\s+', '', val)` — 删除所有空格、制表符、换行符

2. **修复中文冒号**：`val.replace('：', ':')` — 将 `http：//` 修复为 `http://`

3. **过滤无效占位符**：
   - `/`、`\`、`.`、`-` → 返回 `-`
   - `无`、`暂无`、`没有`、`不需`、`未知` → 返回 `-`

4. **过滤非企业网站**（不区分大小写）：
   - `gsxtgov.cn` — 工商注册网站
   - `globalsources.com/domei` — 环球资源平台
   - `shuidi.cn` — 水滴信用
   - `1688.com` — 阿里巴巴1688

5. **检测邮箱误填**：如果值包含 `@`，说明邮箱被误填到网站字段 → 返回 `-`

6. **检测公司名误填**：如果值包含中文字符 `[\u4e00-\u9fff]`，说明公司名被误填 → 返回 `-`

7. **补全 http:// 前缀**：
   - 以 `www.` 开头 → 添加 `http://`
   - 包含 `.` 但不以 `www.` 开头 → 添加 `http://`
   - 不包含 `.` → 返回 `-`（不是有效URL）

8. **域名小写化**：将URL中 `://` 后的域名部分转为小写，路径保持原样

9. **去除尾部斜杠**：`https://xxx.com/` → `https://xxx.com`（仅当无路径时）

## 手机 (clean_mobile)

### 清洗流程

```
原始值 → 检测多号码 → 提取第一个 → 去国际区号 → 去非数字字符 → 校验格式
```

### 详细规则

1. **多号码字段处理**：
   - 检测条件：包含 `\n`、`国内`、`国际` 关键词
   - 提取方式：`re.findall(r'1[3-9]\d{9}', val)` 取第一个匹配

2. **去除国际区号**：
   - `+86` 前缀：`re.sub(r'^\+86\s*', '', val)`
   - `0086` 前缀：`re.sub(r'^0086\s*', '', val)`

3. **去除非数字字符**：`re.sub(r'[^\d]', '', val)`

4. **格式校验**：
   - 标准手机号：`^1[3-9]\d{9}$`（1开头，第二位3-9，共11位）
   - 非标准但11位数字：保留返回
   - 其他：返回清洗后的值或 `-`

## 办公电话 (clean_phone)

### 清洗流程

```
原始值 → 去国际区号 → 保留座机格式
```

### 详细规则

1. **去除国际区号**：同手机字段

2. **座机格式保留**：
   - 如果去除 `-` 和空格后是11位纯数字 → 去除所有 `-` 和空格
   - 否则保留原始格式（如 `0571-12345678`）

## 邮箱 (clean_email)

### 清洗流程

```
原始值 → 修复.cm拼写 → 去空白 → 格式校验
```

### 详细规则

1. **修复拼写错误**：`val.replace('.cm', '.com')` — 常见 typo

2. **去除空白**：`re.sub(r'\s+', '', val)`

3. **格式校验**：`^[^\s@]+@[^\s@]+\.[^\s@]+$`
   - 必须包含 `@`
   - `@` 前后都至少有一个字符
   - `@` 后必须包含 `.`

## 业务联系人 (clean_contact)

仅做首尾空格清理：`val.strip()`

## 扩展规则

如果在使用过程中发现新的脏数据模式，按以下方式扩展：

### 添加非企业网站过滤

在 `04_process_data.py` 的 `non_website_patterns` 列表中添加新模式：
```python
non_website_patterns = [
    ...,
    'newpattern.com',  # 新发现的非企业网站
]
```

### 添加清洗规则

在对应的清洗函数中添加逻辑：
```python
def clean_website(val):
    ...
    # 新增规则：检测其他异常模式
    if '新异常模式' in val:
        return '-'
    ...
```
